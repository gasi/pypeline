pypeline
========

Pypeline encodes and tags your movies & TV shows for iTunes using the latest
data from [IMDb][].


Prerequisites
-------------
  - [Python][]
  - [IMDb.py][]


Installation
------------

    git clone https://github.com/gasi/pypeline

Usage
-----

    python pypeline -s ~/Downloads -d ~/Movies/iTunes/pypeline

Help
-----

    python pypeline --help
